# SyntheticMarketAnalysisTools
A collection of C# and MQL5 projects including the Boom-Crash Analysis Tool, Digit Sniper probability engine, and multiple algorithmic trading EAs. Focused on pattern detection, data processing, automation, and real-time decision-support logic.

Boom-Crash Analysis Tool
A C# desktop application designed for analysing synthetic market behaviour using real-time frequency tracking and volatility-based logic. The tool processes incoming data into structured analysis, identifies price behaviour patterns, and produces decision-support insights.
Features:

Real-time boom/crash pattern evaluation

Rolling-window frequency analysis

Probability-based market behaviour indicators

Clean UI built in Visual Studio (.NET)

Digit Sniper Analysis Tool

A C# probability engine that tracks digit outcomes, calculates conditional probabilities, and identifies high-quality sample windows for decision-making. Build with modular logic and a responsive interface.

Features:

Live digit frequency distribution

Conditional probability engine

Rolling sample window tracking

Built-in logic filters for pattern identification
